📊 Apple Social Media Marketing Analytics using Microsoft Excel
📌 Project Overview

This project analyzes Apple's social media marketing performance across multiple platforms using Microsoft Excel. The objective is to transform raw marketing data into meaningful business insights through data cleaning, calculated metrics, Pivot Tables, Pivot Charts, and dashboard-style analysis.

The project evaluates campaign effectiveness, platform performance, audience engagement, hashtag performance, and follower growth to support data-driven marketing decisions.

🎯 Objectives
Clean and prepare raw social media datasets.
Analyze engagement across different social media platforms.
Measure campaign performance using key marketing metrics.
Identify high-performing hashtags and content types.
Track follower growth trends over time.
Generate actionable insights for marketing strategy.
